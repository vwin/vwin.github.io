–install选项使用了多个参数用于创建符号链接。最后一个参数指定了此选项的优先级，如果我们没有手动来设置替代选项，那么具有最高优先级的选项就会被选中

这个例子中，我们为/usr/bin/python3.5设置的优先级为2，所以update-alternatives命令会自动将它设置为默认Python版本
